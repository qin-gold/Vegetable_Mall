# Vegetable_Mall 购物商城
# 简单介绍
本项目使用Spring+SpringMVC+Mybatis来实现了一个网上商城系统，可以使用maven构建为war包进行部署，也可部署在Tomcat中运行。
# 主要功能
 ## 1.普通用户
      登录、注册功能
			
      浏览商品功能
      搜索商品功能
      查看商品详情
      添加购物车
      购买功能（购物车批量购买）
      查看订单状态
      确认收货功能
      评价已购买商品功能
 ## 2.管理员
      拥有普通用户所有功能
      查看、删除所有用户功能
      查看、删除所有商品功能
      添加新的商品功能
      处理订单功能
      发货功能
